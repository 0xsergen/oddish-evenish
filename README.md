# oddish-evenish
Summing digits of the number and deciding whether it is odd or even

*Sample Usage:*
oddishOrEvenish(12445)

*Working Flow:*

Number: 12445 | Sum: 0

Number: 1244 | Sum: 5

Number: 124 | Sum: 9

Number: 12 | Sum: 13

Number: 1 | Sum: 15

Number: 0 | Sum: 16

Input: 12445 | Sum: 16 | Type: Even
